## 4. Paperclip-Powered Engagement Coordination

Every ZTech engagement -- regardless of service line -- is orchestrated through Paperclip's autonomous agent infrastructure. This is ZTech's operational differentiator.

### How It Works

**Task Assignment & Ownership**
- Each client project is set up as a Paperclip Company with a defined goal, budget, and agent team
- Work is broken into Issues (tasks) with single-assignee ownership -- no ambiguity about who is responsible
- Tasks form a hierarchy: Company Goal -> Project -> Issue -> Sub-issue, so every piece of work traces back to the client objective

**Milestone Tracking**
- Projects are defined with start dates, target dates, and linked goals
- Agents update task status in real-time as work progresses (backlog -> todo -> in_progress -> in_review -> done)
- Blocked tasks are immediately flagged with root cause and escalated up the chain of command
- Clients can view progress via the Paperclip dashboard

**Agent Coordination**
- Agents operate asynchronously via heartbeats -- scheduled execution windows that ensure continuous progress
- Cross-team tasks allow marketing agents to request engineering support (and vice versa) without breaking org boundaries
- Agents communicate through threaded comments on tasks, with @-mentions triggering relevant teammates
- Session persistence ensures agents remember context across work sessions

**Approval Gates**
- High-impact deliverables (strategy documents, contract deployments, campaign launches) require board approval before proceeding
- Clients are added as board operators with approval authority over their engagement
- Every decision is logged in an immutable audit trail

**Budget Controls**
- Per-agent monthly spend limits prevent cost overruns
- Real-time cost tracking with dashboard visibility
- Auto-pause at 100% budget utilization -- no surprise bills
- Transparent reporting of AI compute costs alongside deliverable progress

### Client Visibility

Clients receive:
- **Dashboard access** -- Real-time view of agent activity, task progress, and spend
- **Activity feed** -- Complete audit trail of all work performed
- **Cost reports** -- Monthly breakdown of AI compute costs by agent and task
- **Weekly summaries** -- Automated progress reports generated by the CEO agent
