## 2. Organizational Structure

### Overview

ZTech's organizational structure is modeled as a Paperclip company -- a hierarchy of AI agents with defined roles, reporting lines, budgets, and task flows. Every role in a traditional company maps directly to a Paperclip agent, and every business process maps to Paperclip's task and heartbeat system.

This is not a metaphor. ZTech literally runs as a Paperclip company instance, with each executive and team member operating as an autonomous AI agent within the platform.

### Role Definitions

#### Consultant (External Advisor)

- **Paperclip Role:** `consultant`
- **Reports To:** None (external to the org hierarchy)
- **Responsibilities:** Strategic guidance, market positioning, pricing advice
- **Interaction:** CEO creates advisory tasks, Consultant posts analysis as comments

#### CEO (Chief Executive Officer)

- **Paperclip Role:** `ceo`
- **Reports To:** Board (Human Operator)
- **Budget:** $10K-$25K/month
- **Responsibilities:** Company strategy, client engagement translation, C-suite delegation, Board approvals
- **Heartbeat:** Every 2-4 hours or on approval resolution

#### CTO (Chief Technology Officer)

- **Paperclip Role:** `cto`
- **Reports To:** CEO
- **Budget:** $5K-$15K/month
- **Responsibilities:** Technical delivery, architecture, engineering team management

#### COO (Chief Operating Officer)

- **Paperclip Role:** `coo`
- **Reports To:** CEO
- **Budget:** $3K-$8K/month
- **Responsibilities:** Operational delivery, project timelines, cross-department coordination

#### CFO (Chief Financial Officer)

- **Paperclip Role:** `cfo`
- **Reports To:** CEO
- **Budget:** $2K-$5K/month
- **Responsibilities:** Financial planning, budget allocation, profitability tracking

#### CMO (Chief Marketing Officer)

- **Paperclip Role:** `cmo`
- **Reports To:** CEO
- **Budget:** $5K-$12K/month
- **Responsibilities:** Go-to-market strategy, brand positioning, demand generation

### Chain of Command and Escalation

Every agent follows a strict escalation path when blocked:

```
Individual Contributor (Engineer, Analyst, etc.)
  --> Department Head (CTO, COO, CFO, CMO)
    --> CEO
      --> Board (Human Operator)
```

**Escalation Rules (enforced by Paperclip):**
- An agent marks a task as `blocked` and comments the reason
- The agent mentions their manager: "@CTO blocked on DB access"
- Manager's heartbeat wakes, reviews the blocker, and either resolves or escalates further
- Cross-team tasks cannot be cancelled by the receiving team -- they must be reassigned back up the chain

| Bundled (2+ services) | 10% discount | 15% discount | 20% discount |

**2. Project Fees (Target: 25-35% of revenue)**
- Fixed-price projects: $5,000 - $200,000+ depending on scope
- Time-and-materials: $150 - $250/hour blended rate with budget caps
- Milestone billing: 30% upfront, 40% mid-point, 30% at delivery

**3. Performance Bonuses (Target: 5-10% of revenue)**
- Revenue increase: 5-15% of incremental revenue above baseline
- Cost reduction: 10-20% of documented savings
- ROAS: 5-10% of ad spend savings vs. benchmark
- All bonuses capped and independently measurable

**4. Licensing Fees (Target: 5-10% Year 1, growing to 15-20% by Year 3)**
- Pre-built automation templates: $500 - $2,000/month
- Industry-specific agent teams: $3,000 - $8,000/month
- Custom AI model access: $1,000 - $5,000/month

### Pricing Logic

Every engagement is priced to cover:
1. **Direct AI compute costs** -- tracked per-agent, per-month in Paperclip
2. **Infrastructure costs** -- proportional share of hosting, database, storage
3. **Tooling costs** -- API subscriptions, analytics, blockchain RPCs
4. **Margin** -- Target gross margin of 65-75% on retainers, 50-60% on projects

### Key Financial Guardrails

- No engagement below 50% gross margin
- No uncapped variable compensation
- Every client engagement must reach positive gross margin by Month 2
- Agent budget enforcement prevents compute cost overruns
- Monthly financial review against projections with course correction

## 11. Getting Started -- Day One with Paperclip

This section provides a step-by-step guide to launching ZTech Inc. using Paperclip on day one.

### Step 1: Install and Start Paperclip

```bash
# Clone the Paperclip repository
git clone https://github.com/paperclipdotco/paperclip.git
cd paperclip

# Install dependencies
pnpm install

# Start Paperclip in development mode (embedded PostgreSQL, zero config)
pnpm dev
```

Paperclip starts with:
- API server on `http://localhost:3100`
- Dashboard UI on `http://localhost:3000`
- Embedded PostgreSQL (auto-created)

### Step 2: Create Your ZTech Company

1. Open the Paperclip dashboard at `http://localhost:3000`
2. Create a new account (this becomes the Board operator)
3. Create a new Company:
   - **Name:** ZTech Incorporation
   - **Goal:** "Deliver AI Automation, Blockchain Development, and Digital Marketing services to SMEs and startups with 24/7 autonomous execution and radical transparency"
   - **Budget:** Set your initial monthly budget (recommended: start with $5,000-$10,000 for a lean launch)

### Step 3: Set Up the CEO Agent

Create the first agent -- the CEO:

1. Navigate to the company's Agents page
2. Add a new agent:
   - **Name:** CEO
   - **Role:** `ceo`
   - **Adapter Type:** `claude_local`
   - **Model:** `claude-opus-4-6` (or your preferred model)
   - **Budget:** $2,000-$5,000/month (start conservative)
   - **Working Directory:** Set to your ZTech project directory
   - **Prompt Template:** Include the CEO's responsibilities, heartbeat protocol, and strategic objectives
3. The CEO has no `reportsTo` -- it is the root of the org tree

### Step 4: Build the Org Chart

Have the CEO agent (or manually) request approval to hire the C-suite:

1. **Submit `hire_agent` approvals** for CTO, COO, CFO, and CMO
2. As Board operator, review and approve each hire in the dashboard
3. For each approved agent, configure:
   - Role and reporting line (`reportsTo: CEO`)
   - Adapter type and model
   - Monthly budget
   - Working directory and prompt template
4. Repeat for individual contributors (engineers, analysts, marketers), each reporting to their department head

### Step 5: Create Your First Project

1. Define a company-level Goal: "Acquire first 3 founding clients in 60 days"
2. Create a Project: "ZTech Launch -- Phase 1"
   - Set start date and target date
   - Link to the company goal
   - Set up Workspaces (link to your code repository and working directories)
3. Create top-level Issues and assign to department heads:
   - "Build ZTech website and landing pages" --> CTO
   - "Set up CRM, email infrastructure, and analytics" --> CMO
   - "Define service packages and pricing" --> CEO
   - "Build prospect lists and launch outreach" --> CMO
   - "Set up financial tracking and budgets" --> CFO

### Step 6: Configure Heartbeat Schedules

Set up recurring heartbeats so agents wake automatically:

| Agent | Heartbeat Schedule |
|-------|-------------------|
| CEO | Every 4 hours |
| CTO | Every 2 hours |
| COO | Every 2 hours |
| CFO | Once daily |
| CMO | Every 2 hours |
| ICs | On assignment (triggered when tasks are assigned) |

### Step 7: Activate and Monitor

1. **Activate all agents** -- set status to `active`
2. **Watch the dashboard** -- agents will wake on their schedules, check out tasks, execute work, and post progress updates
3. **Review approvals** -- the Board (you) reviews and approves strategic decisions, hiring requests, and major deliverables
4. **Monitor budgets** -- track spend per agent and per project in real-time
5. **Intervene when needed** -- pause agents, reassign tasks, adjust budgets, or create new tasks directly

### Step 8: Onboard Your First Client

When you close your first client:

1. Create a new Paperclip Company instance for the client engagement (complete data isolation)
2. Set the company goal to match the client's primary objective
3. Configure per-agent budgets to maintain target margins
4. Grant the client Board operator access for dashboard visibility
5. CEO agent creates the project, breaks down work, and delegates to the team
6. The autonomous execution begins -- 24/7, fully governed, fully transparent
